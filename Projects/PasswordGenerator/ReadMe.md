  <h1>Introduction</h1>
  <p>This password generator will generate a password based off of the following options.</p>
  <ul>
  <li>Lowercase letters</li>
  <li>Uppercase letters</li>
  <li>Numbers</li>
  <li>Exclude Duplicates</li>
  <li>Symbols</li>
  <li>Include Spaces</li>
</ul>
